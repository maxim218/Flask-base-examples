# Flask основы

## API - GET запросы

Сумма чисел

```
http://localhost:5000/summa/?a=12&b=5
```

Произведение чисел

```
http://localhost:5000/proizved/?x=12&y=4
```

Добавление элемента в массив

```
http://localhost:5000/push/?v=george
```

Получение всех элементов массива

```
http://localhost:5000/all/
```
